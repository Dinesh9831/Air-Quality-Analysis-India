# Air Quality Analysis – India 🇮🇳🌫️

This project performs an end-to-end data analysis and machine learning workflow on India’s Air Quality Index (AQI) dataset. It includes data cleaning, exploratory data analysis (EDA) with colorful visualizations, supervised learning models, unsupervised clustering techniques, anomaly detection, and result export.

---

## 📌 Project Objectives

- Understand air pollution patterns across different cities and states in India  
- Analyze relationships between geographical and pollutant features  
- Build regression models to predict average pollutant levels  
- Apply clustering techniques to identify pollution-based regions  
- Detect anomalies in air quality data  

---

## 📂 Dataset

- **Source:** data.gov.in
- (Air Quality Index India dataset)
- **Format:** CSV  
- **Key Features:**
  - Country, State, City, Station  
  - Latitude, Longitude  
  - Pollutant Minimum, Maximum, Average  
  - Pollutant ID  
  - Last Update Timestamp
